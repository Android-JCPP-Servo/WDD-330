  • Ladder - "this" JS

Exercise Description

  • Using "this" in object literal - When assigning "this" as a reference inside a function, it is considered undefined. However, when "this" is called outside of a function, or assigned "before the dot", no errors will be generated.

  • Create a calculator - Create an object calculator with three methods:
    • read() prompts for two values and saves them as object properties.
    • sum() returns the sum of saved values.
    • multiply() multiplies saved values and returns the result.

  • Chaining - When coding for a step-ladder, for example, you can combine multiple steps together using "chaining". The idea is that you combine methods together in a sequence, which is then read as if the developer wrote each method individually.
    One example looks like this:

    ladder.up().up().down().showStep();

    Another example looks like this:

    ladder
     .up()
     .up()
     .down()
     .up()
     .down()
     .showStep();
