Chapter 8: Transforms and Transitions

Notes

Chapter 8

  • Transforms and transitions are used to diversify the appearance of a webpage.
  • Many older web browsers use techniques that are similar to transforms and transitions, but the processes to create such details required a great deal of CSS and JavaScript.
  • To simplify the process of creating moveable pictures and text, CSS3 was created and implemented transformations and transitions.
  • If a developer wanted something to move, adjust positioning, or change color on a mouse-hover, the developer could use CSS3's new capabilities to achieve such goals.
  • Additionally, and based on personal experience, transforms and transitions can be used to fix an object's position on the screen. Instead of using "top:" (or "bottom:") and "left:" (or "right:"), a transform can be used to establish the exact same positioning (i.e. "transform: (50px, -50px);")
  • CSS3 transforms and transitions not only reduced the amount of code needed to adjust a picture, but also simplified the coordinate system and provided new special effects for developers.

Chapter 12: Canvas, SVG, and Drag and Drop

Chapter 12

  • Canvas is much similar to an artist's paint canvas, except in a digital manner.
  • When a developer uses Canvas, they are achieving a similar result to creating a shape in a separate platform, like Inkscape.
  • SVG images are files of code that can be used in multiple pages at once, much like a single JavaScript file called multiple times by different individual HTML files.
  • The developer has three ways of creating and displaying SVG images:
    1. Find a picture online, download it as an SVG, and then save the code in a file uploaded to a platform, such as Visual Studio Code.
    2. Draw a picture in Inkscape, save it, download its SVG code, and then upload it to VS Code.
    3. Using Sketch.app:
      1. Upload photos or text to a document
      2. Right-click on the image
      3. Click "Copy SVG Code"
      4. Paste the code to a file
      5. Upload the file to VS Code
  • Finally, Drag and Drop gives the user the ability to move objects with their cursor.
  • Users can either drag and drop files from their computer to their browser (i.e. "Upload Document(s) Here"), or objects from one part of the browser to another part (i.e. an interactive gaming website).
  • Several ways can be used to implement Drag and Drop:
    1. "Set the draggable attribute on any HTML elements you'd like to be draggable.
    2. "Add an event listener for the dragstart event on any draggable elements.
    3. Add an event listener for the dragover and drop events on any elements that you want to have accept dropped items.
